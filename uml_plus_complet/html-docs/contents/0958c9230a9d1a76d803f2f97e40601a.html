enreg debut emprunt

UMLUseCase
bibliotheque :: Model_UseCase :: enreg debut emprunt

Description

scénario nominal:

  1. identifier_abonné (voir sous U.C.)
  2. vérifier droits à emprunter (voir sous U.C.)
  3. pour chaque livre à emprunter:
    • le bibliothécaire enregistre le numero de l'exemplaire à emprunter
    • le système enregistre l'emprunt (lien avec l'abonné , dateDebut, ...)
  4. le système affiche un résumé des opérations effectués
  5. éventuelle impression d'un ticket récapitulatif
